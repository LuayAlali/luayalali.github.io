Guess the Phrase is a game wherein the objective is to guess a common phrase indicated by blank boxes which denote the number of letters.

Languages used: CSS, Vanilla JavaScript.

Read more... Github Live

The user is allocated five lives represented by hearts. When a letter is correctly selected it takes its appropriate position in the blank boxes mentioned above, giving players clues for the remaining letters, and it also prevents user error by disabling any correct letters already "chosen". Alternatively, each incorrect guess results in a loss of a heart. With five incorrect Guesses, the game ends, and the user is prompted to play again.

It's beautiful transitions and constant feedback make it engaging. It also prevents user error by disabiling any correct letters already 'chosen'.
