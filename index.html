Guess is a phrase guessing game. The user is allocated five hearts each right letter guess is displayed, each wrong guess subtracts a heart. If the user gusses wrong five times the game ends and the user is prompted to play again.

Read more... Github Live

Guess is a complete start to finish user experience. If the right letters are chosen the user wins. In the instance of a correct letter selection the letter is showcased in its appropiate location within the word, making it easier for the user to guess the next correct letter.

It's beautiful transitions and constant feedback make it engaging. It also prevents user error by disabiling any correct letters already 'chosen'.
